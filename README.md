# Outpro-India-Website

The Corporate Digital Presence Platform aims to establish a modern, high-performance, and scalable website for Outpro.India, designed to clearly communicate its services, portfolio, team and business capabilities. This project focuses on building visually premium and responsive corporate website that functions as sales -optimized digital identity. 

## How to run

1. **Start the Backend**:
   - Open a terminal in the `server` directory.
   - Run `npm install` and then `node server.js` (Server runs on `http://localhost:5000`).

2. **Start the Frontend**:
   - Open a terminal in the `client` directory.
   - Run `npm install` and then `npm start` (Client runs on `http://localhost:3000`).
